## Команды для проверки

```bash
# Проверка конфигурации
php artisan config:show

# Проверка маршрутов
php artisan route:list

# Проверка кэша
php artisan cache:table
php artisan cache:clear

# Проверка прав доступа
ls -la storage/
ls -la bootstrap/cache/
```
